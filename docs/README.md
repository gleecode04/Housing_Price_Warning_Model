# CS 4641 Project Website

This directory contains the GitHub Pages website for the CS 4641 project "Early-Warning Model for U.S. Housing Price Surges".

## Website Structure

- `index.html` - Main project proposal page
- `styles.css` - Custom CSS styling for modern, professional appearance
- `scripts.js` - JavaScript for interactive features including Gantt chart
- `_config.yml` - GitHub Pages configuration

## Features

- **Responsive Design**: Optimized for desktop, tablet, and mobile viewing
- **Interactive Gantt Chart**: Canvas-based project timeline visualization
- **Smooth Navigation**: Scroll spy and smooth scrolling between sections
- **Word Count Tracking**: Real-time monitoring of proposal word count (800 word limit)
- **Modern UI**: Professional styling with hover effects and animations

## Deployment

This site is configured for GitHub Pages deployment. To deploy:

1. Push the `docs/` folder to your repository
2. In repository settings, enable GitHub Pages from the `docs/` folder
3. The site will be available at `https://[username].github.io/[repository-name]/`

## Content Sections

1. **Introduction/Background** - Literature review and dataset description
2. **Problem Definition** - Problem statement and motivation
3. **Methods** - Data preprocessing and ML algorithms
4. **Results & Discussion** - Metrics, goals, and expected outcomes
5. **References** - IEEE formatted citations
6. **Timeline** - Interactive Gantt chart and contribution table

## Requirements Compliance

The website meets all CS 4641 proposal requirements:
- ✅ Less than 800 words for sections 1-4
- ✅ Literature review with 3+ references
- ✅ Dataset descriptions with links
- ✅ 3+ preprocessing methods identified
- ✅ 3+ ML algorithms identified
- ✅ 3+ quantitative metrics
- ✅ Gantt chart with member responsibilities
- ✅ Contribution table
- ✅ Professional presentation suitable for GitHub Pages
